# Blender-EasyMAT
Blender plugin for simplified creation and application of materials

-----> [Download the Blender plugin from here!](https://github.com/Riccardo-Foschi/Blender-CompactMAT/releases/download/v1.13/EasyMAT.py) <-----

A plugin for Blender that simplifies the creation and application of materials without needing to use nodes.

Main tools:
-create and preview new materials with no meshes selected
-assign or unassign materials to one or multiple meshes with one click
-setup of displacement  with one click
-import multiple textures and assign them to the proper channels with one click
-setup normal or bump map with one click
